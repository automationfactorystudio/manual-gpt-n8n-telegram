🤖 Conecta tu GPT Personalizado con n8n y Telegram

Automatiza el envío de contenido generado por IA directamente a tu Telegram

n8n OpenAI GPT Telegram Automatización

📋 ¿Qué vamos a conseguir?

En este tutorial aprenderás a crear una automatización completa que te permitirá pedirle a tu GPT personalizado que genere contenido y, simplemente diciendo "envía a Telegram", recibirás ese contenido directamente en tu chat de Telegram.


Requisitos previos:

  • Cuenta de ChatGPT Plus (para crear GPTs personalizados)
  • Instancia de n8n funcionando
  • Cuenta de Telegram

🔄 Flujo de la Automatización

👤 Usuario pide contenido al GPT
→
🤖 GPT genera el contenido
→
💬 Usuario dice "envía a Telegram"
→
🔗 GPT llama al webhook de n8n
→
📱 n8n envía a Telegram
1

Crear tu Bot de Telegram

1.1 Busca a BotFather

Abre Telegram y busca @BotFather (es el bot oficial de Telegram para crear bots)

💡 Tip: Asegúrate de que es el BotFather oficial (tiene la verificación azul)

1.2 Crea un nuevo bot

Envía el comando:

/newbot

BotFather te pedirá:

  • Nombre del bot: Por ejemplo "Mi Asistente GPT"
  • Username del bot: Debe terminar en "bot", por ejemplo "miasistente_gpt_bot"

1.3 Guarda tu Token

BotFather te dará un token de acceso HTTP API que se ve algo así:

1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
⚠️ ¡MUY IMPORTANTE! Guarda este token en un lugar seguro. No lo compartas con nadie. Lo necesitarás para configurar n8n.

1.4 Obtén tu Chat ID

Necesitas tu ID personal de Telegram para que el bot te envíe mensajes:

  1. Busca en Telegram el bot @userinfobot
  2. Envíale cualquier mensaje
  3. Te responderá con tu información, incluyendo tu Chat ID (un número)
  4. Guarda ese número, lo necesitarás después
Ejemplo de Chat ID: 123456789

1.5 Inicia conversación con tu bot

Busca tu bot recién creado en Telegram y pulsa Start o envíale cualquier mensaje. Esto es necesario para que el bot pueda enviarte mensajes.

2

Configurar el Workflow en n8n

2.1 Crear un nuevo workflow

Entra a tu n8n y crea un nuevo workflow. Dale un nombre descriptivo como "GPT a Telegram"

2.2 Añadir el nodo Webhook

Busca y añade el nodo Webhook

Configuración del webhook:

  • HTTP Method: POST
  • Path: gpt-telegram (o el nombre que prefieras)
  • Authentication: None (para simplificar)
  • Respond: Immediately
💡 Nota: Una vez configurado, copia la URL del webhook. Se verá algo así:
https://tu-dominio-n8n.com/webhook/gpt-telegram

2.3 Añadir credenciales de Telegram

Antes de añadir el nodo de Telegram, necesitas configurar las credenciales:

  1. Ve a Settings (rueda dentada) → Credentials
  2. Click en New
  3. Busca y selecciona Telegram API
  4. Pega el Token que te dio BotFather
  5. Dale un nombre descriptivo (ej: "Mi Bot GPT")
  6. Guarda las credenciales

2.4 Añadir el nodo Telegram

Conecta un nodo Telegram después del Webhook

Configuración del nodo Telegram:

  • Credential: Selecciona las credenciales que acabas de crear
  • Resource: Message
  • Operation: Send Message
  • Chat ID: Pega tu Chat ID (el número que obtuviste de @userinfobot)
  • Text: Aquí mapeas el contenido del webhook
💡 Mapear el contenido: En el campo "Text", haz click en el icono de expresiones y selecciona: 
{{ $json.content }}

Esto tomará el contenido que envíe el GPT y lo reenviará a Telegram

2.5 Activar el workflow

No olvides activar el workflow usando el toggle en la parte superior derecha

✅ ¡Workflow listo! Tu n8n ya está preparado para recibir contenido del GPT y enviarlo a Telegram
3

Configurar tu GPT Personalizado

3.1 Acceder a la creación de GPTs

Ve a ChatGPT y accede a:

  • Click en tu perfil (esquina superior derecha)
  • Selecciona "My GPTs"
  • Click en "Create a GPT"

3.2 Configuración básica del GPT

Dale un nombre y descripción a tu GPT. Por ejemplo:

  • Nombre: "Creador de Contenido con Telegram"
  • Descripción: "Genera contenido y envíalo directamente a Telegram"

3.3 Añadir las instrucciones

En el campo de instrucciones, añade algo como:

Eres un asistente experto en crear contenido para redes sociales.

Cuando el usuario te pida "envía a telegram", "manda a telegram", 
"envíalo a telegram" o similar, debes usar la acción enviarTelegram 
para enviar el último contenido que generaste.

El contenido debe enviarse tal cual lo generaste, sin modificaciones.

3.4 Configurar la acción (Action)

Ve a la pestaña "Configure" y luego a "Actions"

  1. Click en "Create new action"
  2. En el campo de schema, pega el siguiente JSON:
{
  "openapi": "3.0.0",
  "info": {
    "title": "Enviar a Telegram",
    "version": "1.0.0",
    "description": "Envía contenido generado a Telegram vía n8n"
  },
  "servers": [
    {
      "url": "https://TU-DOMINIO-N8N.com"
    }
  ],
  "paths": {
    "/webhook/gpt-telegram": {
      "post": {
        "operationId": "enviarTelegram",
        "summary": "Envía contenido a Telegram",
        "description": "Recibe contenido y lo envía al chat de Telegram configurado",
        "requestBody": {
          "required": true,
          "content": {
            "application/json": {
              "schema": {
                "type": "object",
                "properties": {
                  "content": {
                    "type": "string",
                    "description": "El contenido a enviar a Telegram"
                  }
                },
                "required": ["content"]
              }
            }
          }
        },
        "responses": {
          "200": {
            "description": "Contenido enviado correctamente a Telegram"
          }
        }
      }
    }
  }
}
⚠️ Importante: Reemplaza https://TU-DOMINIO-N8N.com con la URL real de tu instancia de n8n
⚠️ Verifica la ruta: Asegúrate de que /webhook/gpt-telegram coincida exactamente con el path que configuraste en el webhook de n8n

3.5 Configurar privacidad de la acción

Debajo del schema verás opciones de privacidad:

  • Si es solo para ti: selecciona "Only me"
  • Si quieres que otros lo usen: selecciona la opción apropiada

3.6 Guardar y publicar

Click en "Update" o "Create" para guardar tu GPT

✅ ¡GPT configurado! Tu GPT personalizado ya está listo para enviar contenido a Telegram

Probar tu Automatización

4.1 Primera prueba

Abre tu GPT personalizado y prueba con algo como:

Escribe un post motivacional para Instagram sobre perseverancia

El GPT generará el contenido.

4.2 Enviar a Telegram

Una vez generado el contenido, simplemente escribe:

Envía a telegram

O cualquier variante como:

  • "Manda esto a telegram"
  • "Envíalo a mi telegram"
  • "Telegram"

4.3 Verificar en Telegram

Abre tu Telegram y verifica que has recibido el mensaje de tu bot con el contenido generado

Solución de Problemas

🔧 No recibo mensajes en Telegram

Posibles causas:

  • Verifica que activaste el workflow en n8n (toggle superior derecha)
  • Asegúrate de haber iniciado conversación con tu bot (botón Start)
  • Verifica que el Chat ID sea correcto
  • Comprueba que el token del bot esté bien configurado en n8n

🔧 El GPT no envía nada

Posibles causas:

  • Verifica que la URL en el schema del GPT sea correcta
  • Asegúrate de que el path del webhook coincida exactamente
  • Revisa las ejecuciones en n8n para ver si llegan peticiones
  • Verifica que las instrucciones del GPT sean claras

🔧 Error de conexión en n8n

Posibles causas:

  • Verifica que tu n8n sea accesible públicamente
  • Asegúrate de que no hay restricciones de firewall
  • Comprueba que el certificado SSL esté funcionando

🔧 Revisar logs en n8n

Para diagnosticar problemas:

  • Ve a tu workflow en n8n
  • Click en el menú de tres puntos → "Executions"
  • Revisa las ejecuciones fallidas para ver el error específico

Mejoras y Personalizaciones

💡 Formato mejorado en Telegram

Puedes hacer que los mensajes se vean mejor usando formato Markdown:

En el nodo de Telegram, activa la opción:

  • Parse Mode: Markdown

Y modifica el contenido para añadir formato:

*{{ $json.content }}*

---
✨ Generado por mi GPT personalizado

💡 Guardar en base de datos

Puedes añadir un nodo adicional para guardar todo el contenido generado:

  • Añade un nodo de PostgreSQL, MySQL o la base de datos que uses
  • Guarda el contenido junto con fecha y hora
  • Así tendrás un historial de todo lo generado

💡 Enviar a múltiples destinos

Puedes ramificar el workflow para enviar el contenido a:

  • Email (usando nodo de Email)
  • Google Sheets (para llevar registro)
  • Slack (para equipos)
  • Discord (para comunidades)

💡 Añadir autenticación

Para mayor seguridad, puedes añadir autenticación al webhook:

  • En el nodo Webhook, activa "Header Auth"
  • Define un header secreto (ej: "X-API-Key")
  • Añade ese header en el schema del GPT

💡 Crear diferentes tipos de contenido

Puedes crear varios GPTs especializados:

  • GPT para posts de Instagram
  • GPT para tweets
  • GPT para emails de marketing
  • Cada uno enviando a diferentes canales de Telegram o diferentes destinos
